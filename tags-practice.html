Headings

Headings from `H1` to 'H6' identify blocks and sections Every Page must have 1 H1

Pargraphs

Paragraphs go in P tags

lists

Unordered Lists

Unordered lists are marked with bullet points and go in UL tags. Each list item is in an LI tag

Types of Lists

  • Unordered lists (UL)
  • Ordered lists (OL)

Ordered Lists

Ordered lists are numbered and go in ol tags. Each list item is in an li tag

Anchors aka Links

Anchors tags create links to other pages. The URLs go in the href attribute Go to Duck Duck Go to search for more information

Setting Target= _blank will open the page in a new window See MDN for more details
